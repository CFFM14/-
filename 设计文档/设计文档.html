游戏概述

倒水游戏是一款基于 HTML5 Canvas 的益智解谜游戏，玩家通过点击试管将彩色水倒入目标管中，目标是将所有相同颜色的水集中到同一根试管。

核心玩法

操作流程

  1. 点击试管 → 选中（光晕 + 浮空）
  2. 点击另一根 → 倒水动画启动
  3. 四阶段动画：移动 → 倾斜倒水 → 回正 → 归位
  4. 倒水特效：水流 + 液滴 + 飞溅 + 波纹
  5. 同色集齐 → 瓶盖自动盖上
  6. 全部完成 → 通关!

游戏规则

  • 源管不能为空
  • 目标管必须有空位
  • 目标顶层颜色必须匹配（或为空）
  • 每管默认 4 层（变容 4-7）
  • 全部单色 = 通关
  • 步数计数 + 撤销(Ctrl+Z)
特色功能

🎁 盲盒模式

底部水层被彩虹流彩遮挡，上层倒空后逐层自动揭示，考验玩家记忆力。支持 1-2 层盲盒。

🧪 变容模式

打破固定 4 层容量，每管 2-8 层不等。颜色总数 = 目标管容量，装错即卡关。

✋ 拖拽系统

自由拖拽试管到任意位置，松开后自动弹到最近不重叠空地。

🔧 关卡自制器

可视化编辑 + 盲盒/变容参数调节 + DFS 可解性检测 + 本地存储。

➕ 智能辅助

卡关时随时加空管降难度；颜色悬停提示；瓶盖自动完成标记。

视觉设计

🎨 设计语言

简约清新风格，暖白底色 (#F7F5F0) + 16 色柔和马卡龙色板。直壁圆底玻璃试管，手绘高光与内壁反光。

🌊 动态水特效

动画正弦波浪水面 + 内部涟漪振荡线 + 侧面折射反光椭圆。借鉴 Cocos 游戏引擎实现，纯 Canvas 绘制。

🎬 倒水动画系统

四阶段 1.1 秒动画：移动至目标上方 → 倾斜倒水(手抖扰动) → 回正 → 归位。并行多动画支持。

🌈 盲盒流彩效果

HSL 全色相循环渐变，随时间波动流动，白色 "?" 标记清晰可辨。

技术架构
渲染层

Canvas 2D · 自绘玻璃管 · 水特效 · 四阶段动画 · 拖拽交互 · 波浪水面

逻辑层

创新点

01 · 反向构造关卡生成

从已解状态做随机合法倒水搅乱，天然保证有解，无需事后验证。1 空管极限模式同样适用，搜索效率远高于正向验证。

02 · 盲盒记忆挑战

底部水层彩虹流彩遮挡，上层倒空前不可见/不可操作，逐层揭示增加记忆难度和策略深度。

03 · 变容模式

打破固定 4 层容量，颜色总数 = 目标管容量，装错即卡关，深度策略性远超传统水排序。

04 · 实时水物理特效

动态波浪水面、内部涟漪、侧面反光，借鉴 Cocos 游戏引擎效果，纯 Canvas 2D 实现。

05 · 自制关卡系统

可视化编辑器 + 盲盒/变容参数 + DFS 可解性校验 + localStorage 持久化，玩家即创作者。
